## Modulo node

Per installare questa dipendenza, è necessaria la presenza di un file `package.json` all'interno del proprio pogetto, che può essere creato con questo comando:

```bash
npm init
```

Una volta seguiti i passi della creazione del file, puoi installare il pacchetto, che funzionerà solo se hai già il `token` di accesso a [Nexus][accessing-nexus]:

```bash
npm install --save-dev @maggioli/design-system
```

Se l'installazione è andata a buon fine, all'interno di `package.json`, nelle `devDependencies` dovrebbe trovarsi questo pacchetto node:

<CodeSnippet>
  "@maggioli/design-system": "^{data.version}"
</CodeSnippet>

[accessing-nexus]: /use/configuration/accessing-nexus
