## Configurazione

Il design system è pubblicato come modulo [Node][nodejs] e si presta per essere usato su progetti basati su framework JavaScript come [React][react] o [Angular][angular].

Può essere utilizzato anche senza l'ausilio di framework JavaScript, ma per poterlo installare è necessario installare [Node][nodejs] sulla vostra macchina e preferibilmente appoggiarsi a [nvm][nvm] per gestire più versioni installate, spesso necessarie per lavorare su più progetti contemporaneamente.


[angular]: https://angular.io/
[nodejs]: https://nodejs.org/
[react]: https://it.reactjs.org/
[nvm]: https://github.com/nvm-sh/nvm
