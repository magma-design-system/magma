Il package `mgg-icons` è una font icon library basata su [Google Material Design Icons][gmdi] e [Material Design Icons][mdi].

<PackageInfo package={packageData}/>

---

Per installare il pacchetto `mgg-icons` è necessario eseguire questo comando da terminale:

```bash
npm install --save @maggioli-common/mgg-icons
```

Questo comando installa dentro le dipendenze node questi file:

```javascript
import '@maggioli-common/mgg-icons/build/fonts/mgg-icons.css'
```

Se installato nel posto giusto, il design system dovrebbe riuscire ad includere automaticamente il set di icone per [Storybook][storybook].

Quindi tramite storybook è possibile vedere se tutto funziona da qui:

```bash
npm run storybook
```

Una volta installato il pacchetto
