# Tints and tones

You shoudn't copy the HEX color for SCSS or component development to avoid consistence issues, use the color code instead.

---

### Brand tints

These are the main brand colors, which are used not for states but just for identity pruposes and key actions, related to Maggioli.

<ColorMatrix size="large" filter="brand" colors={Tokens.tint}/>

### Status tints seeds

These are the status colors, they should be used to explain the actions status of the app he’s using,
like a success action, danger, for errors, warnings and infos, basically for **cognitive awarness** of contents they are reading.

<ColorMatrix size="large" filter="status" colors={Tokens.tint}/>

### Tones matrix

To ease the process of making the color palette consistent, tones will be the key to modulate Status and Brand tints with a tone matrix that will be used to adjust their tones when the UI needs a different tone of the same tint seed.

The tones matrix is a generative palette based on the [Scale tool][scale] created by [Hayk An][hayk].
