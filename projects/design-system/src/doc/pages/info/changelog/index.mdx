### Changelog `0.0.71`

In questa versione sono state introdotte funzioni per standardizzare la naming convention delle `props` legate allo stile dei componenti:

- È stata introdotta la nuova sezione [lib][0.0.71/src/lib] che contiene script condivisi tra i componenti
- È stato aggiunto il file [styles.js][0.0.71/src/lib/styles.js] che conterrà tutte le operazioni fatte allo scopo di generare il CSS all'interno dei componenti React
- È stata introdotta la function `appendSelectors` al fine di ripulire da `white space` i selettori vengono generati in produzione
- È stata introdotta la function `globalSelectors` al fine di imporre ai componenti un **coding style** su come vengono definite le pros legate ai selettori CSS generati dallo scaffolding
- È stata introdotta la function `modifiers` al fine di imporre ai componenti un **coding style** su come vengono definiti i modifier BEM all'interno delle props dei componenti
