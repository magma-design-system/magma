The `<mds-label>` web component is the Magma Design System tag/chip used to display a short, non-interactive piece of text such as a status, a category, or a removable selection. It wraps an `mds-text` for its content and optionally an `mds-button` for a delete affordance.

#### Semantic Behavior

- **Text content**: The visible text comes from the `label` prop, which is rendered inside an internal `mds-text` and reflected to the host attribute; the default slot is documented for text only — avoid placing HTML elements or components in it.
- **Deletable affordance**: When `deletable` is set, the component renders a trailing close button (baseline cancel icon) and emits the `mdsLabelDelete` event on activation; the click handler stops propagation so the deletion does not bubble to a surrounding clickable container.
- **Keyboard interaction**: While `deletable`, the internal close button is wired through the shared `KeyboardManager` so it can be triggered by keyboard; this binding is attached on load/update and detached on disconnect and when `deletable` is false.
- **Localization**: The close button's `title` is localized (el/en/es/it); the imperative `updateLang()` method re-resolves the active language from the host.

#### Properties & Visual Configurations

The `variant` and `tone` props follow the shared ladders defined in [`projects/stencil/SPEC.md`](../../../../SPEC.md#tone-and-variant-system); `tone` is limited to the minimal `'strong'` / `'weak'` set and defaults to `'weak'`.

#### Component-specific variants and tones

`variant` accepts both the theme label palette (e.g. `'sky'`, the default) and the status palette (`'error'`, `'warning'`, `'success'`, `'info'`). Pick a status value when the label communicates an outcome or state; pick a theme color when it is a neutral category or decorative tag.

#### Other behavioral props

- **`truncate`** controls overflow handling of the inner text: `'word'` (default) keeps it on one line, `'all'` truncates aggressively, and `'none'` allows multiline wrapping.
- **`typography`** selects a compact type scale (`'caption'`, `'detail'`, `'tip'`) appropriate to the label's small footprint, defaulting to `'caption'`.
