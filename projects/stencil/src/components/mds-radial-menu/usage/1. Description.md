The `<mds-radial-menu>` web component is a floating action menu of the Magma Design System that arranges its child items along a circular arc around a central trigger button, expanding them outward when opened. It is the parent of a compound pair: it owns the trigger and the geometry, while each option is supplied as an `<mds-radial-menu-item>` in the `item` slot.

#### Semantic Behavior

- **Trigger button**: Renders an internal `<mds-button>` whose `tone`, `variant` and `size` are forwarded from the host; clicking it toggles the menu open and closed.
- **Open state**: `opened` is the source of truth for visibility; when set the items animate out along the arc and the trigger icon swaps to a close glyph, and a falsy value is normalized back to `undefined`.
- **Interaction mode**: With `interaction="rightclick"` the trigger is visually hidden and the menu instead opens on a document `contextmenu` event, repositioning the host so the menu is centred on the cursor.
- **Backdrop**: When `backdrop` is enabled an overlay is attached while the menu is open and detached on close, so an outside click can dismiss it.
- **Item registration**: Items are read from the `item` slot on slot change; the component injects each item's positional index and propagates the host `size` down to every `<mds-radial-menu-item>`.
- **Arc geometry**: The host derives whether the sweep is a full circle and whether the start angle exceeds the end angle, driving the direction and distribution of items via CSS custom properties.

#### Properties & Visual Configurations

The internal trigger consumes the shared `variant` / `tone` / `size` ladders defined in [`projects/stencil/SPEC.md`](../../../../SPEC.md#tone-and-variant-system); they default to `'dark'` / `'strong'` / `'lg'` here and are passed straight through to `<mds-button>`.

#### Other behavioral props

- **`angleStart`** and **`angleEnd`** define the angular sweep (in degrees) over which items are distributed; a 360Â° span produces a full ring, while a narrower range produces a fan. Setting `angleStart` greater than `angleEnd` reverses the distribution.
- **`radius`** sets the distance (in `rem`) of the items from the centre, controlling how far they travel when the menu opens.
- **`direction`** chooses whether items lay out `'clockwise'` or `'counterclockwise'` around the arc.
- **`interaction`** selects how the menu is summoned: `'click'` shows the trigger button, `'rightclick'` hides it and binds a context-menu handler at the cursor.
- **`disc`** renders a decorative disc beneath the menu; **`icon`** overrides the default trigger glyph and defaults to the "more-vert" Material icon when unset.
