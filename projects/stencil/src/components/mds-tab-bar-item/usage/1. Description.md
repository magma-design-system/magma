The `<mds-tab-bar-item>` web component is a single selectable tab within the [`<mds-tab-bar>`](../../mds-tab-bar) container. It renders an icon stacked over a text label and acts as one option in the bar's mutually-exclusive selection group, replacing what would otherwise be a bespoke clickable nav/tab element.

#### Semantic Behavior

- **Compound child only**: Must be placed as a direct default-slot child of `<mds-tab-bar>`; it is never used standalone or mixed with other child types, since the parent locates its items via `querySelectorAll('mds-tab-bar-item')` and assigns each one a sequential `id` (`mds-tab-bar-item-{index}`) on load.
- **Parent-coordinated selection**: Clicking the host toggles internal state and, when becoming selected, emits `mdsTabBarItemSelect` carrying its own `id`. The parent listens for this event and re-drives every sibling's `selected` prop so exactly one item is active at a time — items do not deselect each other on their own.
- **Selected state is reflected and watched**: `selected` is `mutable` and `reflect`ed to the attribute, so the parent can set it imperatively; a `@Watch` keeps the internal state in sync whenever the prop changes, and the reflected `[selected]` attribute drives the active color/background styling.
- **Whole-host click target**: The entire host element is the click surface (icon + label), not just the text.
- **Text-only default slot**: The default slot is intended for a plain text string; avoid placing HTML elements or other components inside it.

#### Properties & Visual Configurations

- **`icon`**: Icon name (forwarded to `mds-icon`) shown above the label; use it to give the tab a recognizable glyph. Optional — defaults to empty, in which case no meaningful icon is rendered.
- **`selected`**: Marks this item as the active tab. You typically do not toggle it manually at runtime — the parent manages it — but you can set it on one item as the initial active tab.
- **`typography`**: Controls the label's text style via `mds-text`, constrained to the smaller typography set (`tip` by default, or `option`). Pick `option` for a slightly more prominent label.

The active vs. resting appearance is not configured through props but through the `--mds-tab-bar-item-*` CSS custom properties (background and color, each with a `-selected` variant) listed in `readme.md`.
