# Known issues

---

### Close event triggered in wrong moments

If you trigger the event `onMouseDown` on the slot `top`, then you trigger `onMouseUp` slot `bottom`, the window closes insted of remaining opened.

### Modal is shown while should be hidden

If you set `opened` attribute to `false` as first value before render MdsModal is shown, you should try setting `opened` as `undefined` to avoid first rendering
