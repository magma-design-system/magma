## Configurazione Manuale

Qui si riportano le istruzioni per installare il font con le icone Maggioli e configurare il proprio progetto per l'uso.

Prima di procedere all'installazione è necessario avere configurato il proprio gestore di pacchetti (es.: npm) per accedere al repository Nexus aziendale tramite il file `.npmrc`.

### Installazione pacchetto

Per l'installazione del pacchetto usare il proprio gestore di pacchetti.

Usando npm

```bash
npm install @maggioli-common/mgg-icons
```

Usando yarn

```bash
yarn add @maggioli-common/mgg-icons
```

Usando pnpm

```bash
pnpm add @maggioli-common/mgg-icons
```

### Uso del font

Il modo più facile per usare le icone Maggioli è servire staticamente la
cartella `fonts` e il file `mgg-icons.css` per poi importare quest'ultimo.

```html
<link rel="stylesheet" href="/mgg-icons.css">
```

Esempio di uso delle icone in un file html:

```html
<i class="mgg-icons-status-success"></i>
```

In un progetto Angular con SCSS (grazie a Webpack) è sufficiente aggiungere questa riga al file `styles.scss`.

```scss
@import '@maggioli-common/mgg-icons/mgg-icons.scss';
```

### Altre modalità

In un progetto che fa uso di SASS/SCSS è possibile modificare il path da cui servire i file del font
impostando il valore della variabile `$font-icons-base-url` prima di importare il file SCSS.

```scss
$font-icons-base-url: '/my-assets-folder/font';
@import '@maggioli-common/mgg-icons/mgg-icons.scss';
```

In un sito Angular, per modificare il path da cui servire staticamente i file del font
bisogna modificare il file `angular.json` aggiungendo il seguente oggetto nell'array
del campo `assets`.

```json
{
  "glob": "**/*",
  "input": "node_modules/@maggioli-common/mgg-icons/fonts",
  "output": "my-assets-folder/font"
}
```

Di seguito un esempio completo per un progetto Angular, con i file `angular.json` e `styles.scss`.

```json
{
  "projects": {
    "my-project": {
      "architect": {
        ...
        "build": {
          ...
          "assets": [
            {
              "glob": "**/*",
              "input": "node_modules/@maggioli-common/mgg-icons",
              "output": "assets/font/mgg-icons"
            },
            ...
          ]
        }
      }
    }
  }
}
```

```scss
// styles.scss
$font-icons-base-url: '/assets/font/mgg-icons/';
@import '@maggioli-common/mgg-icons/mgg-icons.scss';
```

## Componente per progetti React

Dentro al Design System, troverai il componente `Icon` fatto appositamente per sfruttare il modulo node `@maggioli-common/mgg-icons`. Dovrai quindi prima installare questo set per utilizzarle.

```bash
npm install @maggioli-common/mgg-icons
```

Mettiamo caso che ti serva un'icona all'interno di un componente React, di basterà utilizzarla in questo modo:


```javascript
import React from 'react'
import Icon from '@Design/Icon/Icon'
import './MyComponent.scss'

const MyComponent = props =>
  <div className="my-component">
    <Icon className="my-component__icon" name="status-warning"/>
  </div>

export default MyComponent
```

Il componente `Icon` si occuperà di decorare il markup per poter renderizzare correttamente l'icona selezionata.
