

# Maggioli design system

This design system is based on [React][react], [Storybook][storybook] and [Gatsby][gatsby], you can see it on [Maggioli design system][storybook-online] website.

The project is currently store in [nexus.maggioli.it][nexus-mgg] under `@maggioli` group.

### Roadmap

There are various of planned changes, that will be added in the next weeks:

1. Prepare the final layout for the design system website (also the local documentation);
2. Create an optional config for the CSS scaffolding generated by SCSS;
3. Due to the complexity which is rising, split the progect in sub projects, something like:
3.1. Design tokens (data behind styles and identity);
3.2. Icons;
3.3. Identity (Maggioli logo with the respective sub projects/services);
3.4. Styles system (SCSS);
3.5. Design system documentation (gatsby, react components and storybook).

This roadmap can change depending on project progression and complexity of the integrations.
