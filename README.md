

# Maggioli design system

This design system is based on [React][react], [Storybook][storybook] and [Gatsby][gatsby], you can see it on [Maggioli design system][storybook-online] website.

The project is currently store in [nexus.maggioli.it][nexus-mgg] under `@maggioli` group.

### Roadmap

There are various of planned changes, that will be added in the next weeks:

1. [Adobe Leonardo][adobe-leonardo] to generate the whole palette from it's config;
2. Prepare the final layout for the design system website (also the local documentation);
3. Create an optional config for the CSS scaffolding generated by SCSS;
4. Due to the complexity which is rising, split the progect in sub projects, something like:
4.1. Design tokens (data behind styles and identity)
4.2. Icons
4.3. Identity (Maggioli logo with the respective sub projects/services)
4.4. Styles system (SCSS)
4.5. Design system documentation (gatsby, react components and storybook)

This roadmap can change depending on project progression and complexity of the integrations.
